# Getting started

This section explains how to customize layouts and views in @@webclient. Only users with layout [customization permissions](../concepts/permissions-and-dependencies.md) have access to customization menus.

### 1. [App-bar menu options](appbar-menu-options.md) 
Explains the App-bar settings in ERP.net Web Client, including interface modes (Standard mode and Advanced mode), layout visibility options, and UI navigation controls.

### 2. [How to customize form](customize-form.md) 
Describes form layout customization in ERP.net Web Client, including rearranging panels, modifying form structure, and saving custom form layouts.

### 3. [How to customize main panel](customize-main-panel.md) 
Covers main panel configuration in ERP.net Web Client, including managing embedded panels, controlling panel visibility, and organizing the primary form workspace.

### 4. [How to customize navigator](../../navigators/guide/customize-nav-panel.md) 
Explains navigator panel customization in ERP.net Web Client, including column selection, column ordering, list layout adjustments, and saving navigator views.

### 5. [How to customize fields](https://docs.erp.net/webclient/introduction/how-to/ui-customization/customize-fields.html) 
Describes field-level UI customization in ERP.net Web Client, including showing or hiding fields, changing field order, and adjusting field display settings.

### 6. [How to add several main panels and tiles](https://docs.erp.net/webclient/introduction/how-to/ui-customization/create-multiple-panels.html) 
Explains how to add multiple main panels and tiles in ERP.net Web Client forms to display parallel data views and enhance layout flexibility
   
### 7. [How to create a Named layout](named-layout.md) 
Explains steps to create and utilize named layouts.
