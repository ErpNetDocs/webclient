# Operations

This section contains instructions on working within the navigators:

### 1. [Navigator features](../../introduction/navigator-features.md)
Learn about the common features available in ERP.net navigators, including inline editing, record access, sorting, hierarchy navigation, grouping, panel menus, row context menus, and other tools for working with data.
### 2. [Customize navigator panel](customize-nav-panel.md)
Learn how to customize a navigator panel by choosing which columns to display, changing their order and size, and saving the customized layout for future use.
### 3. [Use filter row](how-to-use-filter-row-in-navigators.md)
Learn how to use the filter row to quickly filter navigator records by entering values directly below the column headers and combining multiple filter criteria.
### 4. [Bulk fill-down a cell value in a column](fill-down.md)
Learn how to copy a cell value to multiple selected records in the same column, making repetitive data entry faster and more efficient.
### 5. [Create Named layout](../../layouts-and-views/guide/named-layout.md)
Learn how to save your customized navigator layout as a named layout and quickly switch between different layouts whenever needed.
### 6. [Update records](update-records.md)
Learn how to update multiple records at once by preparing the data in a spreadsheet and applying the changes directly from the navigator.
